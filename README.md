# Sublime Snippets for Console Logging

## Location
Clone this repo to:
/Users/[USER_NAME]/Library/Application Support/Sublime Text 2/Packages/User/sublime-console-logs

## Tab Triggers
The following are tab triggers that will produce a styles console log

**Log**
cl

**Key and Value**
clv

**Error**
cle

**Success**
cls

**Header 1**
clh1

**Header 2**
clh1
